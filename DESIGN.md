# Design System for Smart Attendance

**Colors:**

- Primary Accent: Blue (#1565c1 or similar modern blue)
- Backgrounds: Soft White (#F5F8FF), Pure White (#FFFFFF)
- Accents: Red for warnings/errors, Green for success.

**Typography:**

- Font: Inter
- Clean, highly legible, modern sans-serif.

**Aesthetics:**

- Light theme
- Modern, sleek, clean design
- Soft drop shadows for cards
- No sci-fi elements
- Premium user experience

**System Notes for Stitch Generation:**

- Keep designs clean and minimalistic.
- Use soft rounded corners for cards and buttons.
- Do NOT include any physical device frames (no phone frames).
- Follow standard mobile app layout (Top app bar, content, bottom actions).
